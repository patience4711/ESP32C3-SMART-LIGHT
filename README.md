# ESP32C3-SMART-LIGHT
This is software for a very smart led controller with built-in timers and dim function. You can apply it in many cases.
For instance, you can wind the strip around a hollow plastic pipe to create a bulb that you can mount in a lantern or armature.
You can make a very intelligent and low-energy garden light that switches on about sunset and goes out at sunrise. 

The features it has are:
- easy to connect to wifi
- can be controlled via webui, mosquitto and http
- dimmable
- 4 timers with advanced options
- simple hardware
- it logs memorable events
- easy to debug
  
## download
[ESP32-C3-DIMMER](https://github.com/patience4711/ESP32C3-SMART-LIGHT/blob/main/ESP32_C3_DIMMER_v0_2.ino.bin)

## hardware
- esp32c3 super mini
- a mosfet module
- 3mm wide ledstrip 5v
- a 5v power supply
