With this software you can build a very smart led light, complete with voice control. It listens to google-home when you say "hey google, dimmer 20 percent" and has a nice webui.<br>
![frontpage](https://github.com/user-attachments/assets/1c359f68-c641-4b8a-b08e-79f037bf466b)


### application example
I printed a tube, winded a 3mm wide cob ledstrip around it and have the fet module and esp inside. 

### main features
- easy to connect to wifi via espressif rainmaker app
- controllable via web ui, google-home, optional button or http
- 4 advanced timers (a.o. configurable dim level)
